# 🌑 Shivin Singh — Interactive AI/ML Portfolio

> *"I don't get lucky. I make my own luck."*

A sleek, interactive developer portfolio inspired by the aesthetic of **Suits** — deep navy pinstripes, champagne gold accents, and Playfair Display serif typography. Built as a single-file HTML/CSS/JS portfolio with zero dependencies and zero frameworks.

---

## ✨ Features

- **Suits-inspired design** — navy pinstripe background, gold accent system, luxury serif typography
- **12+ project cards** with live category filter (NLP / CV / ML / DL / Data & Analytics)
- **Animated skill bars** — 6 categories, 30+ tools, scroll-triggered fills
- **Timeline experience section** with staggered entrance animations
- **Scroll-aware navigation** — active link highlights as you scroll
- **Fully responsive** — mobile-friendly layout at all breakpoints
- **Zero dependencies** — pure HTML, CSS, and vanilla JS (Google Fonts only)
- **Single file** — deploy anywhere, no build step required

## 📦 Usage

No build step. No npm. No framework.

```bash
# Clone the repo
git clone https://github.com/Shivin00/portfolio.git

# Open directly in browser
open index.html
```

Or just drop `index.html` into any static host:

- **GitHub Pages** — push to `gh-pages` branch
- **Netlify** — drag and drop the file
- **Vercel** — deploy as static site
